# Class 1: Meet Scratch — Project Work

> These are the 4 projects for Class 1. Kids can pick one during free build time, or the instructor can assign them.

---

## Project 1: Greeting Cat
**Goal:** Cat walks and greets the user.

**Instructions:**
1. Start a new Scratch project (the Cat sprite is already there)
2. From **Events**, drag `when green flag clicked`
3. From **Looks**, add `say [Hi! I'm Scratchy!] for (2) seconds`
4. From **Motion**, add `move (100) steps`
5. From **Looks**, add `say [Nice to meet you!] for (2) seconds`
6. Click the Green Flag to test

**Concepts practiced:** Sequence, say block, move block.

---

## Project 2: Spinning Star
**Goal:** Add a star sprite that spins in place.

**Instructions:**
1. Delete the cat sprite (right-click → delete)
2. Click "Choose a Sprite" → search "Star"
3. From **Events**, drag `when green flag clicked`
4. From **Motion**, add `turn right (45) degrees` — repeat this block 4 times by duplicating
5. Click the Green Flag to watch it spin

**Concepts practiced:** Turn block, repetition concept (sets up loops for Class 4).

---

## Project 3: Two Friends Talking
**Goal:** Two sprites have a conversation.

**Instructions:**
1. Keep the Cat sprite
2. Click "Choose a Sprite" → add a Duck
3. Position them on opposite sides of the stage by dragging
4. Click on the **Cat** in the sprite list, then build its script
5. Click on the **Duck** in the sprite list, then build its script
6. Use `wait` blocks to time the conversation

**Concepts practiced:** Multiple sprites with their own scripts, `wait` block for timing.

---

## Project 4: Cat's Big Adventure
**Goal:** Cat moves to different spots on the stage.

**Instructions:**
1. Start with the Cat sprite
2. Use `go to x: y:` to set a starting position
3. Use `glide` blocks to move smoothly between locations
4. Add `say` blocks between movements to narrate the journey
5. Try different x and y values to explore the stage

**Concepts practiced:** `go to` for instant movement, `glide` for smooth movement, x/y coordinates preview.
