Understanding the Scratch Stage: The 2D Coordinate System 5 min

Before we start coding, let's understand how Scratch knows WHERE to put things on the stage. It uses a 2D graph — just like the ones in math class!

The X Axis and Y Axis

  • X axis = the horizontal line (left ↔ right). Goes from -240 (far left) to +240 (far right)
  • Y axis = the vertical line (up ↕ down). Goes from -180 (very bottom) to +180 (very top)
  • Origin (0, 0) = the exact center of the stage
💡 Kid-friendly explanation: "Think of the stage like a treasure map. To find any spot, you need two numbers: how far left or right (x), and how far up or down (y)."

The 4 Quadrants

The X and Y axes divide the stage into 4 sections called quadrants:

🎮 Fun Activity: "Where Am I?"
The instructor calls out coordinates, kids point to where on the stage that would be:
  1. (0, 0) → "Center! Right in the middle."
  2. (200, 150) → "Top-right area — Quadrant 1"
  3. (-200, 150) → "Top-left area — Quadrant 2"
  4. (-200, -150) → "Bottom-left area — Quadrant 3"
  5. (200, -150) → "Bottom-right area — Quadrant 4"
  6. (0, 180) → "Very top, middle"
  7. (-240, 0) → "Far left, halfway up"
Reverse game: Point to a spot on the stage and ask kids to guess the coordinates!
💡 Tip for kids: "If you ever forget, just hover your mouse over the stage — Scratch shows the x and y position at the bottom right!"

Guided Coding 15 min

Build this together step by step:

  1. Click the Events category → drag when green flag clicked
  2. Click Looks → drag say [Hello!] for [2] seconds and snap it below
  3. Click Motion → drag move [10] steps and snap it below
  4. Click the Green Flag — watch the cat say "Hello!" and move
  5. Add turn [15] degrees and another move [10] steps
  6. Click Green Flag again — the cat talks, moves, and turns!
🔑 Key Concept: Blocks run from top to bottom, one after another. This is called sequence.

Free Exploration 10 min

  • Change the number in move to make the cat go farther
  • Change the text in say to anything they want
  • Try adding more blocks from Motion and Looks
  • Try clicking the cat sprite on stage and dragging it around

Wrap-Up & Discussion 5 min

  • "What did you make the cat do?"
  • "What happens if you change the numbers in the blocks?"
  • Preview next class: "Next time we'll make our sprites move in cool ways and change how they look!"

🛠️ Project Work

Project 1: Greeting Cat

Goal: Cat walks and greets the user.

  1. Start a new Scratch project (the Cat sprite is already there)
  2. From Events, drag when green flag clicked
  3. From Looks, add say [Hi! I'm Scratchy!] for (2) seconds
  4. From Motion, add move (100) steps
  5. From Looks, add say [Nice to meet you!] for (2) seconds

Project 2: Spinning Star

Goal: Add a star sprite that spins in place.

  1. Delete the cat sprite (right-click → delete)
  2. Click "Choose a Sprite" → search "Star"
  3. From Events, drag when green flag clicked
  4. From Motion, add turn right (45) degrees — duplicate 4 times

Project 3: Two Friends Talking

Goal: Two sprites have a conversation.

  1. Keep the Cat sprite, add a Duck sprite
  2. Position them on opposite sides of the stage
  3. Build separate scripts for each sprite
  4. Use wait blocks to time the conversation

Project 4: Cat's Big Adventure

Goal: Cat moves to different spots on the stage.

  1. Use go to x: y: to set a starting position
  2. Use glide blocks to move smoothly between locations
  3. Add say blocks between movements to narrate the journey
  4. Try different x and y values to explore the stage
