# 🐱 Scratch Programming for Kids — 7-Class Syllabus

A complete, ready-to-teach Scratch programming course for kids ages 7–12. Each class is a 1-hour session with teaching guides, hands-on projects, and full solutions.

## 📁 Structure

```
scratch-for-kids/
├── class-1-meet-scratch/          # Introduction to Scratch
├── class-2-motion-and-looks/      # Controlling movement & appearance
├── class-3-events-and-sound/      # Interactive projects with events & sound
├── class-4-loops-and-conditions/  # Repeating actions & making decisions
├── class-5-variables-and-score/   # Tracking score, lives, and timers
├── class-6-cloning-and-advanced/  # Clones & combining all concepts
├── class-7-final-project/         # Independent project & show and tell
├── instructor-tips.md             # General teaching tips & FAQ
└── scratch-syllabus.md            # Full syllabus in one file
```

Each class folder contains:
- `class-teaching.md` — Full 1-hour session plan with timing and explanations
- `project-work.md` — 4 project briefs with step-by-step instructions
- `project-solutions.md` — Complete block-by-block solutions
- `class-X.html` — Rich HTML version with inline SVG diagrams (open in browser)

## 🎯 Course Overview

| Class | Topic | Key Concepts |
|-------|-------|-------------|
| 1 | Meet Scratch | Interface, sequence, 2D coordinates |
| 2 | Motion & Looks | Movement, costumes, animation |
| 3 | Events & Sound | Key/mouse events, sounds, broadcasts |
| 4 | Loops & Conditions | forever, repeat, if/else, sensing |
| 5 | Variables & Score | Score, lives, timers, game logic |
| 6 | Cloning & Advanced | Clones, broadcasts, combining concepts |
| 7 | Final Project | Independent build + show and tell |

## 🚀 Getting Started

1. Open any `class-X.html` file in a web browser for the full formatted lesson
2. Or use the `.md` files directly in any markdown viewer
3. All projects use [Scratch 3.0](https://scratch.mit.edu) — free, no install needed
